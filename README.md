A simple Python project to evaluate knowledge base completion (KBC).

## Usage
- requires Python 3.6 or higher

## Evaluation File Format

```
<valid triple>
    head: <concepts space separated>
    tail: <concepts space separated>
<valid triple>
    ...
```
